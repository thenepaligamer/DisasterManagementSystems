# Disaster Management Systems
Disaster management system is a system that can help institutions to better understand disasters and crises by simplifying and improving the way information about incidents are shared with the public. Public will also be well aware of the current inventories and upcoming relief. Users and related authorities are notified if an incident has occurred in a nearby area.
